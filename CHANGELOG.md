# Changelog

All notable changes to the Osaka Jade theme.

## [0.1.0] – 2026-06-12

### Added
- First release of the **Osaka Jade** theme for Cursor / VS Code.
- Complete workbench UI colors built on the authentic jade background `#111c18`.
- Syntax highlighting following the default **Dark+** structure, recolored with
  the Osaka Jade palette (rose strings, sage comments, jade types, gold functions).
- 16 terminal ANSI colors taken 1:1 from the original `alacritty.toml`.
- Semantic highlighting (`semanticTokenColors`) consistent with the TextMate rules.
